### 자율주행 시뮬레이터 (2020.01 ~ 2020.06)
1. 유니티에서 대용량 Raw Sensor 데이터를 전송하기 위한 네트워킹 기법 연구
    > - NetMQ, ProtoBuffer, PlatBuffer
    > - OSI (Open Simulation Interface) 방식으로 데이터를 전송하기 위해서 
    > - Modern C++ - C# Unity 호환 동적 라이브러리 개발
    > 
    > <br/>
    > 
    > 성과
    > - 빠른 속도로 대용량의 센서 데이터 전송이 가능 (1 Lidar / 6 Camera)
    > - 유니티에 호환 가능한 라이브러리 연결 방식 구현, 유연한 코드 작성 가능

2. 가상 센서 모듈의 연구 개발 (Lidar Sensor, Camera Sensor)
    > - Velodyne Lidar 실제 하드웨어 스펙과 동일하게 동작하는 가상 센서 모듈 구현
    > - 다량의 연산으로 인한 시스템 부하를 해결하기 위한 다양한 방법 연구
    > - 유니티 시스템의 한계를 극복하기 위한 기술 연구개발
    >   - Custom & Dynamic Mesh Generation
    >   - GPU Instancing
    >   - ECS, Job System, Burst Compiler
    >   - Computing Shader, GPGPU
    >   - C Module for Unity Plugin
    >   
    > <br/>
    > 
    > 성과
    > - 250만 포인트의 연산 수행 시 렌더링 속도 향상 (40.9 fps / 25.8ms ▶ 83.4 fps / 16.8ms)
    > - 하드웨어 스펙에 따라 옵션을 자유롭게 조절하며 테스트할 수 있는 모듈 구현 (Horizon Vertical FOV, Vertical FOV(Upper, Lower), Angular Resolution, Rotation Rate)
